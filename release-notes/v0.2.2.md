# DaisyUI Snippets 0.2.2

This release turns the extension into a much stronger HTML-first DaisyUI workflow for VS Code.

## Highlights

- Rebuilt the extension around a generated snippet catalog instead of a fragile hand-maintained JSON file.
- Replaced broken pasted examples with valid, editable snippet templates.
- Added a faster and more predictable prefix system.
- Added `!d` for a full HTML starter with DaisyUI and Tailwind browser CDN tags.
- Expanded curated variants for common components like accordion, avatar, checkbox, footer, hero, menu, select, tabs, toast, and toggle.
- Added validation, CI, release checks, GitHub release automation, and Marketplace publishing support.

## Good Starting Prefixes

## Why This Matters

The extension is now more trustworthy in day-to-day use:

- valid markup
- practical defaults
- editable placeholders
- cleaner release discipline

## Notes

- scope remains HTML-only
- MCP, LSP, and framework-specific support are still intentionally out of scope
